== Description ==
This plugin is used to list all the authors based on their recent activity / Post
The plugin also posses to show all the authors 
The plugin also has option to select avatar image sytle


== Installation ==

1. Unzip and Upload the `authors-based-on-recent-posts` folder files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. authors-based-on-recent-posts is installed. 
 
== Frequently Asked Questions ==
 
= Will ABRP blog Authors conflict with my other plugins ? =
ABRP blog Authors works by hooking into Wordpress's widgets system and should play OK with most plugins, however there is no guarantee.
